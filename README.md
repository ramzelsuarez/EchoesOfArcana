## Overview (English)
**Echoes of Arcana** is a 2.5D fantasy JRPG developed as a personal Unreal Engine portfolio project.  
Inspired by classic HD-2D JRPGs, this demo focuses on turn-based combat systems, party mechanics, and basic narrative flow.
The project is designed to showcase game system implementation rather than full-scale content.
